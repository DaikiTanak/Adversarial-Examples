本課題はPython3で実装しました。

・ディレクトリsrcの構成
assign1.py : 課題1のテストが記述されています。
assign2.py : 課題2です。モデルの正解率が出力されます。
assign3.py : 課題3です。敵対的入力をFGSMとBaseline手法により生成し、それぞれに対する正解率を出力します。
assign4.py : 課題4です。FGSMを複数回適用し、それに対する正解率を出力します。
lib.py : 関数をまとめた自作ライブラリ。課題1で実装する関数もこのファイルに含まれています。
Eval.py : モデルの正解率を求める関数accuracyと敵対的入力を生成する関数adv記述されています。
model.py : 予測モデルのクラスを定義したファイルです。
